Things I'm Involved With

Web Security

Browserstats
This is a live browser vulnerability statistics site I built as part of my research in Carnegie Mellon with Collin Jackson. The site collects data via ad networks from thousands of people every day, processes them and presents the results on a vulnerability timeline. It runs on Google App Engine.

We presented this project in Web 2.0 Security & Privacy 2010 conference in Oakland, CA. Find the position paper we presented here.

Browserscope
A community-driven Google project for profiling web browsers. I am contributing JavaScript tests to its security category. It is an open source project.

CMU Secbrowsing Extension
This is a Google Chrome extension that periodically checks vulnerability levels of all your browser plug-ins. It is based on SecBrowsing. I built this extension as part of the Browserstats project. If you are using another browser, you can directly check out the extension page here.

Image Classification

NASA Content Based Planetary Image Search System
A system we are currently building for NASA Ames Research Center. The system is capable of classifying large number of planetary images based on their similarity. It can display similarity based on different features. The backend is built on C++ and Python and uses Hadoop as the distributed computation platform. The final product will support processing and storing over one million images and will be online for the public.

Personal stuff

mea3D
This is a 3D engine I wrote from scratch in JavaScript for fun. All the 3D stuff is pure HTML5, no Flash, no Silverlight, no DirectX and no OpenGL. The engine supports transformations, lighting, loading of custom 3D models in JSON format and has some other cool features. It even has a raytracer which can render reflections and shadows!

Mobile

GameRemote
An Android application which lets you use your phone as a game controller. The app connects to a server where the sensor data is broadcast to other clients. The cool thing is, users can play games in theeir browsers using their phones. No software is needed to on computers!
